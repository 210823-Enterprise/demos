Intro to CSS

CSS can be included in 3 different ways:

  1. Inline Styling with the styleattribute
  2. Internal Styling using the styletag in the head of an html doc.
  3. External Styling using the link element in the head of an html doc.

CSS rules are defined by key/value pairs, separated by a colon { : }
